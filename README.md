# Free Map

一个基于 Vue 3、Vite、TypeScript 和高德地图 Web JS API 的私有租房找房地图工具。

这个项目适合用来做自己的“租房观察地图”：你可以限定一个私有找房范围，在地图上记录候选小区的优点、缺点、是否明确淘汰，再结合附近地铁、公交和步行路线辅助判断。

## 功能介绍

- 全屏地图 + 悬浮操作面板，尽量把视口留给地图本身。
- 默认内置上海 11 号线嘉定北到南翔一段的找房范围和重点站点展示。
- 支持框选新的私有找房范围，并一键聚焦回当前范围。
- 支持地图点选创建房源 Marker，记录：
  - 优点
  - 缺点
  - 是否属于“垃圾小区，不能租用”
- 支持编辑和删除已有 Marker。
- 支持地点搜索，结果会按距离最近 11 号线站点的远近排序。
- 支持从搜索结果直接创建房源 Marker。
- 支持搜索当前选中房源附近的地铁站和公交站。
- 支持步行路线规划：
  - 从房源到附近 POI
  - 从房源到你在地图上手动点击的目的地
- 支持导出 / 导入 JSON 配置。
- 支持本地自动保存 Marker 和搜索历史。

## 使用场景

- 筛选某一条通勤线路附近的租房候选点。
- 记录多个小区的优缺点，避免反复看房后信息混乱。
- 在手机或电脑地图上快速判断离地铁、公交、步行通勤的便利程度。
- 把自己的找房范围和标记导出成 JSON，留作备份或迁移到另一台设备。

## 环境要求

- Node.js 18+
- pnpm 10+
- 可用的高德 Web JS API Key

## 环境变量

项目启动前需要准备高德地图相关配置。当前代码会读取下面两个环境变量：

```env
VITE_AMAP_KEY=高德地图的密钥
VITE_AMAP_SECURITY_JS_CODE=如已开启安全密钥请填写
```

可以写到项目根目录的 `.env` 中。

注意：

- `VITE_AMAP_KEY` 必填，没有它地图无法初始化。
- `VITE_AMAP_SECURITY_JS_CODE` 是否需要，取决于你的高德控制台安全配置。
- 不要把真实 Key 和安全密钥提交到仓库。

## 快速开始

```sh
pnpm install
pnpm dev
```

启动后默认会监听：

```text
http://0.0.0.0:5173
```

你可以在本机浏览器里打开 Vite 输出的实际访问地址进行使用。

## 基本操作

### 1. 搜索地点

- 在顶部搜索框输入地点关键字。
- 搜索结果会显示：
  - 编号
  - 地点名称
  - 最近地铁站
  - 到最近地铁站的距离
- 点击结果项会让地图居中到该地点。
- 点击“创建Marker”可以直接把该地点转成待编辑房源标记。

### 2. 手动添加房源标记

- 直接点击地图中的合法位置。
- 弹窗中填写优点、缺点，并可勾选“垃圾小区，不能租用”。
- 保存后 Marker 会出现在地图和左上角列表中。

### 3. 编辑或删除标记

- 点击地图上的 Marker 可以编辑。
- 点击左上角列表项可以选中并居中到对应房源。
- 可以在列表中删除，也可以点击地图 Marker 右上角删除按钮删除。

### 4. 调整找房范围

- 点击“框选范围 R”按钮，或直接按键盘 `R`。
- 在地图上拖出一个矩形范围。
- 框选完成后点击“确定”才会真正替换当前范围。
- 范围确认后，范围外 Marker 会被移除。

### 5. 看附近配套和路线

- 先在左上角列表里选中一个房源 Marker。
- 点击“附近地铁”或“附近公交”。
- 点击搜索到的 POI，可以规划从房源到该 POI 的步行路线。
- 也可以点击“选点路线”，再直接点地图上的任意目的地来规划路线。

### 6. 导出和导入数据

- 点击“保存 JSON”导出当前地图配置。
- 点击“导入 JSON”恢复之前导出的配置。

导出的内容包括：

- 当前地图名称
- 当前找房范围
- 所有房源 Marker

## 自动保存说明

当前版本会自动保存：

- 房源 Marker
- 搜索历史

不会自动保存：

- 当前私有范围
- 当前路线结果
- 当前 POI 搜索结果

如果你改过找房范围，或者准备换浏览器、换设备、移动文件位置，建议主动使用“保存 JSON”做一次备份。

如果浏览器禁用了 `localStorage`，页面会提示你改用“保存 JSON / 导入 JSON”来持久化数据。

## 构建

### 常规构建

```sh
pnpm build
```

### 本地单文件构建

```sh
pnpm build:local
```

这个命令会生成：

- `dist-local/free-map.html`

它会把 CSS 和 JS 内联进一个 HTML 文件，适合本地直接打开使用。

### macOS 压缩包构建

```sh
pnpm build:mac
```

这个命令会进一步生成：

- `free-map-mac.zip`

适合在 macOS 上打包后分发给自己或熟人直接打开。
